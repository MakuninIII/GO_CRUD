# CRUD API для музыкальных групп

Данный проект представляет собой RESTful API для управления музыкальными группами на языке Go с использованием базы данных SQLite. С помощью этого API можно выполнять операции создания, чтения, обновления и удаления (CRUD) записей о музыкальных группах.

## Возможности

- Добавление новой музыкальной группы
- Получение списка групп с поддержкой фильтрации и сортировки
- Обновление информации о группе
- Удаление группы
- Авторизация с помощью JWT-токена

## Авторизация

Для доступа к защищённым маршрутам используйте JWT-токен, который можно получить по эндпоинту `/token`.
